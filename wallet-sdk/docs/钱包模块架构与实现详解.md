## 一、架构总览

### 1.1 分层架构

TrustAid 钱包模块采用**四层架构设计**，自底向上分别为：

```
┌─────────────────────────────────────────┐
│         UI 组件层 (Components)           │
│  ConnectWalletModal（统一连接入口）      │
│  NetworkSwitcher, WalletStatusBadge     │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│       React Hooks 层 (useWallet)         │
│  状态管理、事件订阅、自动重连            │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│      前端适配器层 (wallet-adapter)       │
│  单例管理、配置管理、交易历史            │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│      钱包 SDK 层 (@trustaid/wallet-sdk)  │
│  核心客户端、加密存储、签名服务          │
└─────────────────────────────────────────┘
```

### 1.2 设计原则

| 原则 | 描述 | 实现方式 |
|------|------|----------|
| **隐私第一** | 私钥永不出端 | 本地加密存储、密码解锁、日志脱敏 |
| **用户主权** | 用户完全控制数据 | 本地存储、可导出助记词、可删除钱包 |
| **安全优先** | 多重安全防护 | AES 加密、HD 钱包、审批流程 |
| **体验优化** | 无感连接、自动重连 | 状态持久化、定时器管理、错误重试 |
| **合规审计** | 所有操作可追溯 | 结构化日志、交易历史、错误码映射 |

### 1.3 支持的钱包模式

#### Injected 模式（注入式钱包）

- **适用场景**: MetaMask、Rabby、Coinbase Wallet 等浏览器插件
- **连接方式**: `window.ethereum` 注入 provider
- **账户管理**: 依赖插件自身账户管理
- **签名方式**: 插件弹窗签名
- **网络切换**: 支持 `wallet_switchEthereumChain`

#### Embedded 模式（内置钱包）

- **适用场景**: 隐私优先、无需安装插件
- **连接方式**: 本地加密存储 + 密码解锁
- **账户管理**: HD 钱包派生、助记词导入
- **签名方式**: 本地私钥签名（密码解锁后）
- **网络切换**: 不支持（需重建客户端）

## 二、钱包 SDK 核心实现

### 2.1 客户端创建 (`client.ts`)

#### 2.1.1 `createWalletClient` 函数

**功能**: 创建钱包客户端单例，支持自动/注入/内置三种模式

**源码位置**: `wallet-sdk/src/client.ts#L39-L98`

**实现逻辑**:

```typescript
export function createWalletClient(options: WalletClientOptions = {}): WalletClient {
  // 1. 模式选择（默认 auto）
  const mode = options.mode || "auto";
  
  // 2. 创建注入式钱包客户端
  const injected = new InjectedProviderClient(
    options.providerResolver?.() ?? resolveInjectedProvider()
  );
  
  // 3. 创建内置钱包密钥库
  const embedded = makeEmbedded(options);
  
  // 4. 解析运行时（auto/injected/embedded）
  const runtime: WalletRuntime = resolveRuntime(
    mode, 
    injected.hasProvider()
  );
  
  // 5. 创建服务层
  const injectedSvc = createInjectedService(injected);
  const embeddedSvc = createEmbeddedService(embedded, options);
  
  // 6. 返回客户端接口
  return {
    getRuntimeInfo,
    requestAccounts,
    signMessage,
    signTypedData,
    sendTransaction,
    switchChain,
    addChainThenSwitch,
    // Embedded 专属方法
    createEmbeddedWallet,
    importEmbeddedMnemonic,
    deriveNextEmbeddedAccount,
    listEmbeddedAccounts,
    selectEmbeddedAccount,
    subscribe
  };
}
```

**关键参数**:

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `mode` | `"auto" \| "injected" \| "embedded"` | `"auto"` | 钱包模式 |
| `storageKey` | `string` | `"trustaid_wallet_sdk"` | LocalStorage 键名 |
| `mnemonicMode` | `"mnemonic-persisted" \| "mnemonic-ephemeral"` | `"mnemonic-persisted"` | 助记词存储策略 |
| `rpcUrl` | `string` | - | 默认 RPC 节点地址 |

**运行时解析逻辑** (`resolveRuntime`):

```typescript
// auto 模式：优先 injected，无则 embedded
if (mode === "auto") {
  return hasInjectedProvider ? "injected" : "embedded";
}
// 显式指定模式
return mode;
```

#### 2.2.2 核心方法详解

##### `create(password: string): Promise<string>`

**功能**: 创建新的随机钱包

**实现步骤**:

1. 生成随机 HD 钱包：`HDNodeWallet.createRandom()`
2. 提取助记词：`wallet.mnemonic?.phrase`
3. 初始化状态：
   ```typescript
   const state: EmbeddedState = {
     accounts: [{ address: wallet.address, privateKey: wallet.privateKey }],
     selectedAddress: wallet.address,
     mnemonic: mnemonic, // 根据配置决定是否存储
     nextAccountIndex: 1
   };
   ```
4. 加密并存储：`saveStateEncrypted(password, state)`
5. 返回地址：`wallet.address`

**安全要点**:
- ✅ 私钥和助记词使用 AES-GCM 加密
- ✅ 密码不存储，仅用于加解密
- ✅ 助记词根据 `mnemonicMode` 配置决定是否持久化

##### `importMnemonic(password: string, mnemonic: string): Promise<string>`

**功能**: 导入现有助记词创建钱包

**实现步骤**:

1. 验证助记词：`HDNodeWallet.fromPhrase(phrase)`（失败则抛出 `MNEMONIC_INVALID`）
2. 派生第一个账户（路径 `m/44'/60'/0'/0/0`）
3. 初始化状态并加密存储
4. 返回地址

**错误处理**:

```typescript
try {
  const wallet = HDNodeWallet.fromPhrase(phrase);
  // ...
} catch (e) {
  throw mapWalletError(e, WalletErrorCodes.MNEMONIC_INVALID, "Invalid mnemonic phrase");
}
```

##### `deriveNextAccount(password: string): Promise<string>`

**功能**: 派生下一个 HD 账户

**实现步骤**:

1. 解锁钱包：`unlock(password)` 获取状态
2. 获取助记词：`getMnemonicForDerive(state)`（未存储助记词则抛出错误）
3. 计算派生路径：`m/44'/60'/0'/0/${accountIndex}`
4. 派生新钱包：`HDNodeWallet.fromPhrase(phrase, undefined, path)`
5. 检查是否重复，添加至账户列表
6. 更新 `nextAccountIndex` 并保存

**HD 钱包派生规则**:

```
主路径：m/44'/60'/0'/0/n
- 44': BIP44 标准
- 60': 以太坊链 ID（十六进制 0x3c）
- 0': 账户索引（TrustAid 固定为 0）
- 0: 外部链（接收地址）
- n: 地址索引（从 0 开始递增）
```

##### `unlock(password: string): Promise<EmbeddedState>`

**功能**: 解密钱包状态

**实现**:

```typescript
async unlock(password: string): Promise<EmbeddedState> {
  return this.loadStateDecrypted(password);
}

private async loadStateDecrypted(password: string): Promise<EmbeddedState> {
  const env = await this.readEnvelope(); // 读取加密信封
  if (!env) throw new WalletSdkError(WALLET_NOT_FOUND, "Embedded wallet not found");
  return decryptJson<EmbeddedState>(env.encrypted, password); // AES 解密
}
```

##### `signMessage(password: string, message: string): Promise<string>`

**功能**: 签名消息

**实现步骤**:

1. 解锁钱包获取状态
2. 获取当前选中账户的私钥
3. 创建 `Wallet` 实例：`new Wallet(selected.privateKey)`
4. 签名：`wallet.signMessage(message)`
5. 返回签名结果

**安全要点**:
- ✅ 私钥仅在内存中短暂存在
- ✅ 签名后立即释放（GC 回收）
- ✅ 密码验证通过后才可访问私钥

##### `sendTransaction(password: string, rpcUrl: string, tx: TxRequestV1): Promise<string>`

**功能**: 发送交易

**实现步骤**:

1. 解锁钱包获取私钥
2. 创建 RPC 提供者：`new JsonRpcProvider(rpcUrl)`
3. 创建签名钱包：`new Wallet(selected.privateKey, provider)`
4. 构建交易参数：

5. 发送交易：`wallet.sendTransaction(...)`
6. 返回交易哈希：`sent.hash`

### 2.3 加密模块 (`crypto.ts`)

#### 2.3.1 `encryptJson` 函数

**功能**: 使用 AES-GCM 加密 JSON 数据

**实现**:

```typescript
export async function encryptJson<T>(data: T, password: string): Promise<KeystorePayload> {
  // 1. 生成随机盐值（32 字节）
  const salt = crypto.getRandomValues(new Uint8Array(32));
  
  // 2. 从密码派生密钥（PBKDF2, 100000 次迭代）
  const key = await deriveKeyFromPassword(password, salt);
  
  // 3. 生成随机 IV（12 字节）
  const iv = crypto.getRandomValues(new Uint8Array(12));
  
  // 4. 序列化数据
  const encoded = new TextEncoder().encode(JSON.stringify(data));
  
  // 5. AES-GCM 加密
  const encrypted = await crypto.subtle.encrypt(
    { name: "AES-GCM", iv },
    key,
    encoded
  );
  
  // 6. 返回加密载荷
  return {
    salt: Array.from(salt),
    iv: Array.from(iv),
    ciphertext: Array.from(new Uint8Array(encrypted))
  };
}
```

**安全参数**:

| 参数 | 值 | 说明 |
|------|-----|------|
| 盐值长度 | 32 字节 | 防止彩虹表攻击 |
| PBKDF2 迭代 | 100,000 次 | 增加暴力破解成本 |
| IV 长度 | 12 字节 | AES-GCM 标准长度 |
| 加密模式 | AES-GCM | 提供认证加密 |

#### 2.3.2 `decryptJson` 函数

**功能**: 解密 JSON 数据

**实现**:

```typescript
export async function decryptJson<T>(payload: KeystorePayload, password: string): Promise<T> {
  // 1. 从载荷提取盐值和 IV
  const salt = new Uint8Array(payload.salt);
  const iv = new Uint8Array(payload.iv);
  
  // 2. 从密码派生密钥（使用相同盐值）
  const key = await deriveKeyFromPassword(password, salt);
  
  // 3. AES-GCM 解密
  const decrypted = await crypto.subtle.decrypt(
    { name: "AES-GCM", iv },
    key,
    new Uint8Array(payload.ciphertext)
  );
  
  // 4. 反序列化并返回
  const text = new TextDecoder().decode(decrypted);
  return JSON.parse(text) as T;
}
```

**错误处理**:

- 密码错误：抛出 `CRYPTO_ERROR`（解密失败）
- 数据损坏：抛出 `STORAGE_CORRUPTED`（JSON 解析失败）

### 3.1 单例管理 (`wallet-adapter.ts`)

#### 3.1.1 `getWalletClient` 函数

**功能**: 获取或创建钱包客户端单例

**实现逻辑**:

```typescript
let singleton: ReturnType<typeof createWalletClient> | null = null;
let isInitializing = false;

export function getWalletClient() {
  if (singleton) return singleton; // 已存在直接返回
  if (isInitializing) {
    console.warn('[Wallet] 正在初始化中，请稍候...');
    return null; // 防止重复初始化
  }

  isInitializing = true;
  
  try {
    // 创建客户端
    singleton = createWalletClient({ 
      mode: currentMode, 
      storageKey: "trustaid_wallet_sdk",
      mnemonicMode: "mnemonic-persisted"
    });
    
    // 更新运行时状态
    const runtime = singleton.getRuntimeInfo().runtime;
    useAuthStore.getState().setWalletRuntime(runtime);
    
    // 订阅钱包事件
    if (!unsubscribed) {
      singleton.subscribe({
        onAccountsChanged: (accounts) => { /* ... */ },
        onChainChanged: (chainIdHex) => { /* ... */ },
        onDisconnect: (error) => { /* ... */ }
      });
      unsubscribed = true;
    }
    
    return singleton;
  } catch (error) {
    singleton = null;
    isInitializing = false;
    throw error;
  } finally {
    isInitializing = false;
  }
}
```

**设计要点**:

- ✅ **单例模式**: 避免重复创建客户端导致状态不一致
- ✅ **初始化锁**: `isInitializing` 防止并发初始化
- ✅ **事件订阅**: 仅订阅一次，避免重复监听
- ✅ **状态同步**: 自动更新全局 store

### 3.3 账户管理

#### 3.3.1 `requestPrimaryAccount` 函数

**功能**: 请求主账户（优先弹出钱包选择器）

**实现逻辑**:

```typescript
export async function requestPrimaryAccount(): Promise<string> {
  const client = getWalletClient();
  if (!client) throw new Error("钱包客户端未初始化");
  
  try {
    // 1. 优先尝试请求账户（injected 模式）
    const accounts = await client.requestAccounts();
    const addr = String(accounts[0] || "");
    if (!addr) throw new Error("未取得地址");
    
    useAuthStore.getState().setWalletSessionState("unlocked");
    console.log('[Wallet] 已连接:', formatAddress(addr));
    return addr;
  } catch (error) {
    // 2. 如果是 embedded 模式且有密码，尝试列出账户
    if (currentMode !== "injected" && embeddedConfig.password) {
      const accounts = await client.listEmbeddedAccounts(embeddedConfig.password);
      const addr = String(accounts[0] || "");
      if (!addr) throw new Error("embedded 钱包不存在账户，请先创建或导入");
      
      useAuthStore.getState().setWalletSessionState("unlocked");
      console.log('[Wallet] 已连接 (embedded):', formatAddress(addr));
      return addr;
    }
    
    throw error;
  }
}
```

**降级策略**:

```
Injected 模式成功 → 返回账户
        ↓
    失败（无插件）
        ↓
```

**`connect` 函数实现**:

```typescript
const connect = useCallback(async () => {
  // 1. 防止重复连接
  if (isConnectingRef.current) {
    console.warn('[useWallet] 正在连接中，请稍候...');
    return;
  }

  isConnectingRef.current = true;
  setIsConnecting(true);
  clearError();
  setWalletSessionState('connecting');

  // 2. 设置连接超时（30 秒）
  connectTimeoutRef.current = setTimeout(() => {
    const timeoutError = new Error('连接超时，请检查网络或钱包插件');
    const mappedError = mapWalletError({ code: 'TIMEOUT', message: timeoutError.message });
    setError(mappedError.message);
    setLocalError(mappedError.message);
    setWalletSessionState('failed');
    setIsConnecting(false);
    isConnectingRef.current = false;
  }, CONNECT_TIMEOUT_MS);

  try {
    // 3. 请求主账户
    const addr = await requestPrimaryAccount();
    
    // 4. 连接成功，清除定时器
    clearTimers();
    setRetryCount(0);
    
    console.log('[useWallet] 连接成功:', formatAddressUtil(addr));
  } catch (err) {
    // 5. 清除超时定时器
    if (connectTimeoutRef.current) {
      clearTimeout(connectTimeoutRef.current);
      connectTimeoutRef.current = null;
    }

    // 6. 映射错误
    const mappedError = mapWalletError(err);
    const errorMessage = mappedError.message;
    const errorCode = mappedError.uiCode;

    console.error('[useWallet] 连接失败:', errorMessage);

    // 7. 用户拒绝不记录错误
    if (!isUserRejected(err)) {
      setError(errorMessage);
      setLocalError(errorMessage);
    }

    // 8. 网络错误可以尝试重连
    if (isNetworkError(err) && retryCount < MAX_RETRY_COUNT) {
      console.log('[useWallet] 网络错误，准备重试...');
      reconnectTimerRef.current = setTimeout(() => {
        setRetryCount(prev => prev + 1);
        connect();
      }, RECONNECT_INTERVAL_MS);
    } else {
      setWalletSessionState('failed');
    }

    throw mappedError;
  } finally {
    setIsConnecting(false);
    isConnectingRef.current = false;
  }
}, [clearError, setError, setWalletSessionState, setIsConnecting, clearTimers, retryCount]);
```

**连接状态机**:

```
初始状态 (locked)
      ↓
  用户点击连接
      ↓
连接中 (connecting) ←──┐
      ↓                │
  请求账户            超时/错误
      ↓                │
  成功 → 已解锁 (unlocked)
      ↓
  用户断开/异常 → 已锁定 (locked) + 自动重连
```

**`handleDisconnect` 函数**:

```typescript

  setAddress(null);
  setWalletChainId(null);
```

```
    ↓
```

### 7.2 错误映射函数

#### 7.2.1 `mapWalletError` 函数

```typescript
export function mapWalletError(
  error: unknown,
  defaultCode: string = WalletErrorCodes.UNKNOWN_ERROR,
  defaultMessage?: string
): { uiCode: string; uiMessage: string } {
  // 1. 已经是 SDK 错误
  if (error instanceof WalletSdkError) {
    const mapped = ERROR_CODE_MAP[error.code] || {
      uiCode: error.code,
      uiMessage: error.message
    };
    return mapped;
  }
  
  // 2. 普通错误
  if (error instanceof Error) {
    const message = error.message.toLowerCase();
    
    // 根据关键词匹配
    if (message.includes('password')) {
      return ERROR_CODE_MAP['INVALID_PASSWORD'];
    }
    if (message.includes('mnemonic')) {
      return ERROR_CODE_MAP['MNEMONIC_INVALID'];
    }
    if (message.includes('network') || message.includes('rpc')) {
      return ERROR_CODE_MAP['NETWORK_ERROR'];
    }
    
    // 默认处理
    return {
      uiCode: defaultCode,
      uiMessage: defaultMessage || error.message
    };
  }
  
  // 3. 未知错误
  return {
    uiCode: defaultCode,
    uiMessage: defaultMessage || '未知错误，请稍后重试'
  };
}
```
